写在前面

闲来无事，突发奇想。不知不觉已在技术的道路上摸索了几个年头，才发现还没有认真总结过。 于是在这里收集了一路上陪我走来的各类资源，以便大家交流学习。

资源列表

高级编程语言

C/C++

C Programming Language by Brian W. Kernighan abd Dennis M. Ritchie， 俗称 K&R | C语言的经典之作，其内容覆盖的不仅仅是C语言的语法，更包含这个奇妙而又怪异的语言背后深邃的内涵。

进年来支持文言文编程，汉语编程的环境已逐步成熟，还出现了Zhanyong Wan利用Python二次开发的以东北方言为关键字的编程语法 dongbei 。然而笔者想在这里说一句，在我们享受这些开源工作者提供的便利时，应该留意思考隐藏在语法背后的信息处理原则。 不管是编程语言，自然语言还是数学语言，一切产生意义的信号，本身只有其物理性质。
"言者所以在意，得意而忘言。" - 《庄子·外物》

Dev C++ | Windows 平台的C/C++开发工具，历史悠久，开源集成。

GCC Tool for Windows | 这年头一般都用集成IDE了，不过没有IDE的时候安装了这个也能写码。居安思危，熟悉command line和notepad，关键时刻永远处变不惊。

Python

PySerial Python的串口库，方便调试数据可视化

MuPDF 可以非常便捷的用python操作PDF文件，支持读取修改目录，书签，标注，剪辑，合并等等。

Click 用python轻松创建一个属于你的命令行交互界面 （不知道什么时候会用到这个， 也许你有一天会觉得某些terminal看起来不顺眼，那就是大干一场的时候了，到时请记得分享）。

PyWin32 提供windows强大的软硬件接口。 Windows出10了，API越来越高端了，软件开发也有UWP了，但像笔者这样从底层折腾起来的人，还是win32的API用着顺手，简单粗暴。

ECDict 某位高人为python制作的包含340万词汇量的中英词典。340万什么概念，就是你看着不像单词的词这里面都收录了。-_-!

Pillow Python的一个图像处理工具，用openCV也行，我一般一块儿用。

MicroPython

官网 用python开发单片机，很有想法。网上的介绍：MicroPython本身使用GNU C进行开发，在ST公司的微控制器上实现了Python 3的基本功能，拥有完善的解析器、编译器、虚拟机和类库等。 在保留了python语言主要特性的基础上，他还对嵌入式系统的底层做了非常不错的封装，将常用功能都封装到库中，甚至为一些常用的传感器和硬件编写了专门的驱动。 我们使用时只需要通过调用这些库和函数，就可以快速控制LED、液晶、舵机、多种传感器、SD、UART、I2C等，实现各种功能，而不用再去研究底层模块的使用方法 （说了半天其实我也没用过）。

业余无线电

软件定义无线电 SDR

AirSpy SDR 比较常用，自带ADS-B，支持Windows和Linux。

GNU Radio GNU的开源项目，不必多说，旗下有社区维护的各类无线电软硬件设备和网络系统。给个 中文社区 用来快速上手。

硬件

RTL-SDR 这是个有故事的电视棒，官网上有详细说明。接收范围24~1700MHz，绕过Tuner可以采样到500KHz。

USRP 通用软件无线电外设（Universal Software Radio Peripheral）。旨在使普通计算机能像高带宽的软件无线电设备一样工作。 从本质上讲，它充当了一个无线电通讯系统的数字基带和中频部分。（瞅着不赖，溜去查查价格）

无线电网络

OpenBTS 开源项目，旨在打造一个基于软件定义无线电的通讯网络，可使用GNU Radio + USPR 配合GSM空中接口和VoIP实现的低廉GSM通信网络

APRS 自动位置回报系统（Automatic Packet Report System） 为业余无线电中的一个项目， 结合业余无线电和全球卫星定位系统（GPS）以AFSK AX.25协议达到即时位置传送的目的。 因为是分布式网络，没有中心节点，也没有数据共享，主要靠各地爱好者架设天线和中继器。于是 APRS-IS （APRS互联网服务）应运而生。 在以下这些服务器平台上可查看周围网络节点。

  • aprs.fi平台： https://aprs.fi/
  • HelloCQ APRS平台： http://aprs.hellocq.net/


用户可使用开源软件Dire Wolf 搭配上文硬件中提到的无线收发器器轻松建立Digipeater，iGate, 以及Beacon。项目页面里有详细的介绍。

考试

业余无线电考试练习网站 可以在手机、平板、电脑上练习, 并且支持自动记录错题,便于着重复习。同时还能记录整体练习进度, 是业余无线电考试的好帮手。

写在最后

强烈推荐看下 这个网站， 网站充斥着f**k，sh*t等不雅词汇，其实是在用这种特异的风格传达一种设计哲学，引人深思。网站内容预览如下：

用这位大哥自己的话总结就是这样：

"Good design is as little design as possible."
- some German motherfucker
笔者看到它的时候正在做本网页的收尾工作，说来惭愧。在本页面的设计上虽说处处偷工减料，但是也花了些功夫，尤其是logo的设计。 当时被这句话戳中痛点，突然间意识到，没有花样繁复的容器，信息仍就是信息。

曾经为了使声音通过电线传播，为了把信息发到大洋彼岸，多少前人苦心搭建起信息系统，自己却在百无聊赖地为字体选颜色，这是一个多么糟糕的时代啊。 越来越多的晶体管，越来越高的带宽，可究竟多少处理和传输的信息具有实际意义，而不只是随风而逝的热量和光。

经典的Unix哲学一时深入人心，然而若干年后，技术的成熟和生产力的过剩再次在不可阻挡的时代潮流中服务起了人类无尽的审美与物质需求。 我们不止一次地站在历史的转折点凝望这个在苦难中挣扎了几千年的文明，满心欢喜地看着它逐步踏上理性的道路，却不料历史一次次重新上演。

最后，让我们回味一下20世纪走过的路，并了解一下 DARPA 这个站在几乎一切当代技术背后的投资者。希望这篇文章能带给你一些想法。
